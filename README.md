<h1> Tutorial de Utilização e Instalação da Aplicação </h1>


Roda o sistema localmente:


<strong> 1. Clonar o repositório </strong>

<strong> 2. Abra a pasta e insira no terminal</strong> 

```console
npm install
```

<strong> 3. Abra o .env e adicione a sua do senha Mysql (sua senha) no campo BD_SENHA e o nome do seu Database no campo BD:</strong>
```console
BD = "ADICIONE SUA DATABASE AQUI"
BD_SENHA = "ADICIONE SUA SENHA AQUI" 
```

<strong> 4. Crie a DataBase no seu banco (MySQL):</strong>

```console
CREATE DATABASE exercicio;
```

<strong> 5. Rode o script Mysql disponibilizado no repositorio para criar as tabelas:</strong>

<br>

<strong> 6. Rodar a aplicação:</strong>

```console
node index.js
```
